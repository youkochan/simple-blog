136 Single Number

除一个元素外，每个元素出现两次。找出这个元素。无注意事项，按异或做。线性时间复杂度，可以不使用额外空间。

int singleNumber(int A[], int n) {
        while(--n > 0){
            A[0] ^= A[n];
        }
        return A[0];
    }

137 Single Number II

除一个元素外，每个元素出现三次。找出这个元素。线性时间复杂度，可以不使用额外空间。每次循环先计算 twos，即出现两次的 1 的分布，然后计算出现一次的 1 的分布，接着 二者进行与操作得到出现三次的 1 的分布情况，然后对 threes 取反，再与 ones、twos进行与操作，这样的目的是将出现了三次的位置清零。

int singleNumber(int A[], int n) {
	    int ones = 0;
	    int twos = 0;
	    int threes = 0;
	    
	    while (--n >= 0) {
	        twos |= ones & A[n];
	        ones ^= A[n];
	        threes = ones & twos;
	        ones &= ~threes;
	        twos &= ~threes;
	    }
	    return ones;
	}

54 Spiral Matrix

Given a matrix of m x n elements (m rows, n columns), return all elements of the matrix in spiral order. 螺旋形输出数组，用四个变量记录上下左右四个方向可以达到的最远位置（即边界线），然后循环输出即可，每次换方向时更新边界线的值。特殊一点的情况在于多行一列的矩阵，若遍历过程中不达到边界线则会造成死循环（在当时那个错误的代码环境下）。通过的代码如下，应该能有更简洁的写法。

int *spiralOrder(int** matrix, int numRows, int numColumns) {
	    int* result = NULL;
	    int top = 0;
	    int right = numColumns - 1;
	    int left = 0;
	    int bottom = numRows - 1;
	    int count = 0;
	    int total = numRows * numColumns;
	    int i = 0, j = 0;
	    
	    result = (int*)malloc(sizeof(int) * total);
	    
	    while(count < total){
	        while(j <= right && count < total){ result[count] = matrix[i][j]; count++; j++;}
	        top ++;
	        j--;
	        i++;
	        while(i <= bottom && count < total){ result[count] = matrix[i][j]; count++; i++;}
	        right --;
	        i--;
	        j--;
	        while(j >= left && count < total){ result[count] = matrix[i][j]; count++; j--;}
	        bottom --;
	        j++;
	        i--;
	        while(i >= top && count < total){ result[count] = matrix[i][j]; count++; i--;}
	        left ++;
	        i++;
	        j++;
	    }
	    
	    return result;
	}

59 Spiral Matrix II

和上一题类似，要求生成类似的矩阵。代码都差不多的啦~

int **generateMatrix(int n) {
	    int** matrix = NULL;
	    int i = 0, j = 0;
	    int count = 0;
	    int total = n * n;
	    int top = 0;
	    int right = n - 1;
	    int left = 0;
	    int bottom = n - 1;

	    matrix = (int**)malloc(sizeof(int*)*n);
	    for(i=0; i<=n-1; i++) {
	    	matrix[i] = (int*)malloc(sizeof(int)*n);
	    }

	    i = j = 0;

	    while(count < total){
	        while(j <= right && count < total){ count++; matrix[i][j] = count; j++;}
	        top ++;
	        j--;
	        i++;
	        while(i <= bottom && count < total){ count++; matrix[i][j] = count; i++;}
	        right --;
	        i--;
	        j--;
	        while(j >= left && count < total){ count++; matrix[i][j] = count; j--;}
	        bottom --;
	        j++;
	        i--;
	        while(i >= top && count < total){ count++; matrix[i][j] = count; i--;}
	        left ++;
	        i++;
	        j++;
	    }

	    return matrix;
	}

135 Candy

There are N children standing in a line. Each child is assigned a rating value.

You are giving candies to these children subjected to the following requirements:

Each child must have at least one candy. Children with a higher rating get more candies than their neighbors.

一开始没注意是邻居，于是，需要最少的糖果的话，即满足“糖果数目从1开始，每次不得不增加时，仅增加一颗”。使用长度为n的数组p保存每一个孩子最终得到多少糖果，初始化为每一个元素都为1。

  1. 先从左至右遍历，遍历过程中，若ratings[i]>ratings[i-1], 则p[i]=p[i]+1。遍历完成后得到一个糖果分配方案，保证了从左至右满足题目的要求。
  2. 再从右至左遍历，遍历过程中，依旧从t=1开始，若ratings[i]>ratings[i+1]，则p[i]=max(p[i], t)，t++。若不满足条件，则跳过。
  3. 经过两次遍历后的数组的和即为所求。
int candy(int ratings[], int n) {
	    int*p = NULL;
	    int s = 0;
	    int i = 0;
	    int t = 0;

	    p = (int*)malloc(sizeof(int)*n);

	    for(i = 0; i <= n-1; i++){
	    	p[i] = 1;
	    }

	    for(i = 1; i <= n-1; i++){
	    	p[i] = ratings[i] > ratings[i-1] ? p[i-1] : 1;
	    }

	    s += p[n-1];
	    t = 1;
	    for(i = n-2; i >= 0; i --){
	    	if(ratings[i] > ratings[i+1]){
	    		t += 1;
	    		p[i] = max(p[i], t);
	    	}
	    	else{
	    		t = 1;
	    	}
	    	s += p[i]; 
	    }

	    return s;
	}

26 Remove Duplicates from Sorted Array

Given a sorted array, remove the duplicates in place such that each element appear only once and return the new length.

Do not allocate extra space for another array, you must do this in place with constant memory.

For example, Given input array A = [1,1,2],

Your function should return length = 2, and A is now [1,2].

简单，考虑特殊情况，用一个pos位置记录需要求改的位置，一次遍历即可。

int removeDuplicates(int A[], int n) {
	    int i = 0;
	    int pos = 0;
	    
	    if(n == 0)
	        return 0;
	        
	    if(n == 1)
	        return 1;
	    
	    for(i; i <= n-1; i++) {
	        if(i == 0 || A[i] != A[i-1]) {
	            A[pos] = A[i];
	            pos ++;
	        }
	    }
	    
	    return pos;
	}

6 ZigZag Conversion

按照折线的方式将输入字符串转化成另一种按行输出的格式。思路是按行输出，找准每一行的第i个元素在原数组中的对应座标。

代码如下：

char *convert(char *s, int nRows) {
	    char* l = NULL;
	    int i = 0;
	    int size = strlen(s);
	    int pos1 = 0;
	    int pos2 = 0;
	    int flag = 0;

	    l = (char*)malloc(sizeof(char)*(size+1));

	    if(nRows == 1){
	        for(i = 0; i <= size-1; i++)
	            l[i] = s[i];
	        l[i] = '\0';

	        return l;
	    }

	    for(i = 0; i <= nRows-1; i++) {
	        pos1 = i;
	        if(i == 0 || i == nRows-1){
	            while(pos1 <= size - 1) {
	                l[pos2++] = s[pos1];
	                pos1 = pos1 + 2*(nRows - 1);
	            }
	        }
	        else{
	            flag = 1;
	            while(pos1 <= size - 1) {
	                l[pos2++] = s[pos1];
	                pos1 += flag ? 2 * (nRows - i - 1) : 2 * i;
	                flag = !flag;
	            }
	        }
	    }

	    l[pos2] = '\0';

	    return l;
	}
