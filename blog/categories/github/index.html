静态博客搭建

总结一下最近这几天折腾的博客搭建吧(github+octopress)

环境

  • OS: OS X Yosemite 10.10
  • Ruby: ruby 2.0.0p481 (2014-05-08 revision 45883)
  • git: git version 1.8.5.2 (Apple Git-48)
  • xcode: xcode command line tool

环境配置上面纠结了我很长时间, octopress要求最低ruby版本为1.9.3, mac自带版本为2.0.0, 可以直接使用. 可是由于之前对ruby一无所知, 所以在环境配置上出现了许多乱七八糟的问题. 最后发现是由于ruby的默认的源http://rubygems.org/经常不稳定, 于是可以通过淘宝镜像来配置源. 代码如下:

gem source -a http://ruby.taobao.org/
gem source -r http://rubygems.org/

这样就能替换掉自带的源(→_→ 此处耽误了我N小时). 至于xcode command line tool用于gcc以及make之类操作, Mac OS X 10.9 Mavericks中的Xcode中已经没有选项来自动安装之, 于是我们可以通过终端来安装:

xcode-select --install

git在mac os也是自带的, 到此, 我们便完成了静态博客的环境配置.

github配置

首先你需要一个github账号(废话), 然后在你的账号设置中, 选择新建一个repository. 注, 新建的repository的名字必须为(yourname).github.com, 此处的yourname为你想要的域名. 此时, github的配置告一段落.

octopress配置

首先下载最新版的octopress: github https://github.com/imathis/octopress, 下载安装的教程可以参见网页的readme文件.
命令如下:

git clone git://github.com/imathis/octopress.git ~/octopress
cd octopress
gem install bundler
bundle install
rake install

此时便完成了octopress的安装. 此时, 输入

rake preview

便可以通过localhost:4000来预览你的博客(需要保证你的4000端口不被占用)

关联github

创建密匙(若.ssh目录下已经有密匙可以跳过)

ssh-keygen -t rsa -C username

按照提示完成后, .ssh目录下应该有id_rsa及id_rsa.pub这两个文件, 打开id_rsa.pub, cmd+a, cmd+c, 复制下来.
打开这个网址https://github.com/settings/ssh, add ssh key, title随便写, key的话就把刚刚id_rsa.pub内的东西黏贴进去. add, 搞定233333.

发布到github

终端输入

rake setup_github_pages

会要求你输入github仓库的地址, 此时把之前创建的repository的地址输入即可. 如git@github.com:your_username/your_username.github.io.git

终端输入

rake generate   # 生成HTML页面
rake deploy     # 发布HTML页面

到此, 你就可以在你的(yourname).github.com看到你的博客啦(当然, 什么都没写的话什么都没有的, 而且自带主题真是…搓).

终端输入

rake new_post["titile"] # zshell记得在[]左边加\

这样会在source/_post目录下生成一个Markdown文件, 利用Markdown编辑软件或者文本编辑软件即可修改. 修改完后输入上面发布以及生成页面的指令便可以发布你刚刚写的blog到github上啦~~
