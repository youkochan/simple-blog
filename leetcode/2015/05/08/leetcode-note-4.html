199 Binary Tree Right Side View

Given a binary tree, imagine yourself standing on the right side of it, return the values of the nodes you can see ordered from top to bottom.

题目大意是，给你一个二叉树，你站在这棵树的右边，把看到的序列从顶到底输出。一开始我想错了，我以为简单地自顶向下，若有右孩子，则向右移动指针；若没有则左移。后来发现这个简单的想法是错误的。因为有可能左子树的高度大于右子树，因此，向左看的时候看到的最下面一部分是左子树。

由于需要输出一个数组，则可以使用一个buffer，buffer[i]表示i层能看到的数字。

因此，考虑在中序（中右左）递归中记录一个遍历到的最大深度（初始为0），若当前遍历的节点深度大于或者等于最大深度，则将该节点加入buffer（因为在这种情况下该节点一定是最右边的节点，因为遍历顺序的原因，先中，再右，再左）；若深度小于等于最大深度，说明右边有更高的子树（肯定会挡住这个节点，因此这个节点不可见），则跳过这个点，继续搜索这个节点的子节点。

代码如下：

/**
    * Definition for a binary tree node.
    * struct TreeNode {
    *     int val;
    *     struct TreeNode *left;
    *     struct TreeNode *right;
    * };
    */
    /**
    * Return an array of size *returnSize.
    * Note: The returned array must be malloced, assume caller calls free().
    */
    
    void searchTree(struct TreeNode* node, int buffer[], int level, int* maxLevel) {
        if(node == NULL)
            return;
            
        if(level >= (*maxLevel)) {
            buffer[level] = node->val;
            (*maxLevel) += 1;
        }
        
        searchTree(node->right, buffer, level+1, maxLevel);
        searchTree(node->left, buffer, level+1, maxLevel);
    }
    
    int* rightSideView(struct TreeNode* root, int* returnSize) {
        int buffer[2000];
        int*result = NULL;
        int count;
        int*maxLevel = (int*)malloc(sizeof(int));
        
        (*maxLevel) = 0;
        
        searchTree(root, buffer, 0, maxLevel);
        
        count = (*maxLevel);
        
        if(count == 0) {
            (*returnSize) = 0;
            
            return NULL;
        }
        else {
            result = (int*)malloc(sizeof(int)*count);
            (*returnSize) = count;
            
            for(count = count-1; count >= 0; count--) {
                result[count] = buffer[count];
            }
            
            return result;
        }
    }
