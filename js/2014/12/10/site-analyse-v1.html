网站结构分析v1版以及v2版

在宿舍学习javascript的时候有了一个想法，既然最近不是得探索探索可视化之类的东西，平日浏览的网页结构不应该也可以可视化出来么？

抱着这个想法，就琢磨了一下。

javascript:(function(){
	script=document.createElement('script');
	script.setAttribute('src','http://youkochan.github.io/assets/bookmarklet/siteAnalyse.vx.js');
	document.getElementsByTagName('head')[0].appendChild(script);
})();

代码采用javascript实现，具体使用方式为，打开浏览器，输入以上代码（有的浏览器不支持自动将空格转化为％20需手动替换一下，vx替换成v1或者v2）
