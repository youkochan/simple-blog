121 Best Time to Buy and Sell Stock I

通过一次交易，实现利润最大。通过一次遍历，使用一个变量记录遍历当前最低价格，使用一个变量记录当前最大差价。一次遍历即可。代码如下：

int maxProfit(int prices[], int n){
    int i;
    int low = -1;
    int max = 0; 

    for(i = 0; i<= n - 1; i++){
        if(low < 0 || prices[i] < low)
            low = prices[i];
        if(prices[i] - low > max)
            max = prices[i] - low;
    } 
    return max;
}

122 Best Time to Buy and Sell Stock II

通过多次交易，实现利润最大。这个由于交易次数不限，所以只需要把所有后一天比前一天增加的价格求和就可以了。一次遍历，代码如下：

int maxProfit(int prices[], int n){
    int i = 0;
    int max = 0;
    int d = 0;

    if(n == 1)
        return 0;

    for(i = 1; i <= n - 1; i++){
        d = prices[i] - prices[i-1];
        if(d > 0)
            max += d;
    }

    return max;
}

123 Best Time to Buy and Sell Stock III

通过两次交易，实现利润最大化。由于交易次数有限，可以简化为将prices划分为两个区间，通过求两个区间分别进行一次交易得到的利润和的最大值，来求解这一题。一般情况下，先遍历0到n-1进行划分，再求划分区间的一次交易最大利润（即Best Time to Buy and Sell Stock I，复杂度O(n)），总复杂度O( n2 )。
通过使用动态规划算法可以简化复杂度，使用两个长度为n的数组记录中间结果，dp1[i]表示[0,...,i]中进行一次交易的最大利润，dp2[i]表示[i,...,n-1]中进行一次交易的最大利润，则，可知递推公式如下：

dp1[i+1] = max( dp1[i], prices[i+1]-minPrice )
dp2[i-1] = max( dp2[i], maxPrice-prices[i-1] )

其中， minPrice以及maxPrice分别记录递归过程中的最小以及最大值。代码如下：

int max(int n1, int n2){
    return n1 > n2 ? n1 : n2;
}

int maxProfit(int prices[], int n)
{
    int* dp1;   // [0,...,i] 最大收益
    int* dp2;   // [i,...,n-1] 最大收益
    int maxprofit = 0;
    int i;
    int minPrice = prices[0];
    int maxPrice = prices[n-1];

    dp1 = (int*)malloc(sizeof(int)*n);
    dp2 = (int*)malloc(sizeof(int)*n);

    for (i = 0; i <= n-1; ++i){
        dp1[i] = dp2[i] = 0;
    }

    if (n <= 1)
        return 0;

    // 第一趟遍历找出dp1
    for (i = 1; i <= n-1; ++i){
        if (minPrice > prices[i])
            minPrice = prices[i];
        dp1[i] = max(dp1[i-1], prices[i]-minPrice);
    }

    // 第二趟遍历找出dp2
    for (i = n-2; i >= 0; --i){
        if (maxPrice < prices[i])
            maxPrice = prices[i];
        dp2[i] = max(dp2[i+1], maxPrice-prices[i]);
    }

    // 最后找出最大收益
    for (i = 0; i <= n-1; ++i){
        if (dp1[i] + dp2[i] > maxprofit)
            maxprofit = dp1[i] + dp2[i];
    }

    free(dp1);
    free(dp2);

    return maxprofit;
}

188 Best Time to Buy and Sell Stock IV

再上题基础上，不限制交易次数，设交易次数为k，求最大利润。需要使用动态规划来处理。可以维护两个数组用来存放中间结果，即

  1. global[i][j]，表示到第i天为止，最多进行j次交易可以的到的最大利润；
  2. local[i][j]，表示到第i天为止，且第i天卖出股票（即获得收益），且最多进行k次交易可以得到的最大利润。

通过分析可以得出以下递推公式：

global[i][j] = max( global[i-1][j], local[i][j] )
local[i][j] = max( global[i-1][j-1] + max( diff, 0 ), local[i-1][j] + diff )

其中diff为prices[i]-prices[i-1]，具体解释如下：

①全局到i-1天进行j-1次交易，然后加上今天的交易（如果今天的交易赚钱的话）。
②取局部第i-1天进行j次交易，然后加上今天的差值（ local[i-1][j]是第i-1天卖出的交易，它加上diff后变成第i天卖出，并不会增加交易次数。无论diff是正还是负都要加上，否则就不满足local[i][j]必须在最后一天卖出的条件了 ）

还有一点需要注意的是，题目中的交易次数k有可能远大于天数n，当k远大于n时，问题转化为不限次数求最大利润，即II题的答案。在本题中，可认为k>=n时，问题IV转化为问题II，代码如下：

int max(int n1, int n2){
    return n1 > n2 ? n1 : n2;
}

int** getMatrix(int n, int k){
    int i = 0, j = 0;
    int** p = NULL;

    p = (int**)malloc(sizeof(int*)*n);
    for (i = 0; i <= n-1; i++){
        p[i] = (int*)malloc(sizeof(int)*k);

        for (j = 0; j <= k-1; j ++){
            p[i][j] = 0;
        }
    }
    return p;
}

void freeMatrix(int** p, int n){  // 不可用！会导致超时
    int i = 0;

    for (i = 0; i<= n-1; i++){
        free(p[i]);
    }
    free(p);
}

int simpleMaxProfit(int prices[], int n){
    int i = 0;
    int max = 0;
    int d = 0;

    if(n == 1)
        return 0;

    for(i = 1; i <= n - 1; i++){
        d = prices[i] - prices[i-1];
        if(d > 0)
            max += d;
    }

    return max;
}

int maxProfit(int k, int prices[], int n)
{
    int** g = NULL;
    int** l = NULL;
    int i = 0;
    int j = 0;
    int diff = 0;

    if(n == 1)
        return 0;
    if(k >= n / 2)
        return simpleMaxProfit(prices, n);

    g = getMatrix(n, k+1);
    l = getMatrix(n, k+1);

    for (i = 1; i<= n-1; i++){
        diff = prices[i] - prices[i-1];
        for (j = 1; j <= k; j ++){
            l[i][j] = max(g[i-1][j-1] + max(diff, 0), l[i-1][j]+diff);
            g[i][j] = max(g[i-1][j], l[i][j]);
        }
    }

    return g[n-1][k];
}
