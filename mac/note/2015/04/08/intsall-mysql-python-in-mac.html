在mac上安装Python模块MySQLdb

最近由于有必要使用MySQLdb，而使用pycharm自带的包管理界面进行安装时发现无论如何都安装失败，于是采用了手动安装的方式。具体步骤如下：

1 下载 MySQL for Python。链接，或者采用以下代码

wget http://sourceforge.net/projects/mysql-python/files/mysql-python/1.2.3/MySQL-python-1.2.3.tar.gz

2 解压执行python setup.py build若出现EnvironmentError: mysql_config not found问题则应该首先修改setup_posix.py文件。将mysql_config.path = "mysql_config"修改为：mysql_config.path = "/usr/local/mysql/bin/mysql_config"

3 执行（可能需要sudo）：

python setup.py build
python setup.py install

4 在执行完上述步骤后仍然可能出现import MySQLdb时的错误，此时需要执行以下指令：

sudo ln -s /usr/local/mysql/lib/libmysqlclient.18.dylib /usr/lib/libmysqlclient.18.dylib
